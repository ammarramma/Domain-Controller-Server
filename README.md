# Domain Controller Server

A complete guide to building a Windows Server infrastructure from scratch — including Active Directory, networking services, web hosting, and remote access.

## Quick Highlights

| Service | Purpose |
|---------|---------|
| **SConfig / IP Setup** | Server foundation — static IP, hostname, DNS |
| **ADDS** | Active Directory domain `A3N4.com` |
| **DHCP** | Automatic IP assignment (scope: `172.18.18.11` – `.20`) |
| **DNS** | Forward & reverse lookup zones |
| **Join Domain** | Client registration to domain |
| **User & Group** | OU, security groups, user accounts |
| **IIS** | Web server with HTTPS |
| **IIS - Laravel** | PHP/MySQL web app deployment |
| **Remote Desktop** | Remote server management |
| **ADFS** | Federation services with custom theme |
| **DFS** | Distributed file system with NTFS permissions |
| **FSRM** | File quota management |
| **NTP** | Time synchronization |
| **FTP** | File transfer server |
| **Mail Server** | MailEnable email server |
| **VPN** | Remote access VPN |
